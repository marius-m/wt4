Ticket code

This is a ticket code that worklog belongs to.

Ex.: 'DEV-123'

- When cursor is active in the field, there is a list of recently saved tickets

- Press magnifying glass icon to open search screen

- Press on copy icon it will copy ticket code http link (useful when pasting in browser)

Tips and tricks

- You can paste in whole link, it will pick up ticket code automatically

Recent tickets screen has filter to help out finding tickets

- Search by ticket description

- Filter by ticket number (Ex.: '111' - would find 'DEV-111')

- Filter out by ticket code (Ex.: 'DEV' - would find 'DEV-111')

- You can change worklog screen size by dragging on screen corner

Keyboard shortcuts

- Press CMD / CTRL + ENTER saves the worklog

- Press ESC closes the worklog screen

- Press ALT + TAB to cycle through input fields

- Press ALT + SHIFT + TAB to cycle through input fields in reverse

- Press CMD / CTRL + F opens ticket search screen
